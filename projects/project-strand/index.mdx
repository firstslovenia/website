# Project Strand
A project that aims to standardize robot parts for FTC. <br/>
Parts in this library are being developed by teams #22903 VegaMind and #28080 VegaTech.

## Mecanum drivetrain
This is a four wheel mecanum drive based drivetrain, compatible with both the REV Robotics and goBILDA build systems. <br/>
